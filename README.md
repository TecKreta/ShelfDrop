# ShelfDrop 📦

**マウスを端まで動かすだけで現れる、フローティングクリップボード棚。**  
ドラッグ&ドロップでファイルを一時保管し、すぐに取り出せます。

---

## 主な機能

- 🖱️ **マウスシェイクで呼び出し** — 画面の端でカーソルをシェイクするだけで棚が出現
- 📁 **ファイル棚** — 画像・PDF・どんなファイルでも一時保存してドラッグで取り出し
- 📝 **プロンプト棚** — テキストやプロンプトをコレクション。ワンクリックでコピー＆自動ペースト
- 🔢 **複数の棚** — 用途別に棚を複数作成可能
- 💾 **自動保存** — アプリを終了しても棚の中身は保持

## インストール方法

### 通常の手順

1. `ShelfDrop.zip` をダウンロードして解凍
2. `ShelfDrop.app` を `/アプリケーション` フォルダへ移動
3. アプリを起動

### ⚠️ 「開発元を確認できません」と表示された場合

macOS の Gatekeeper によるセキュリティ確認です。以下のいずれかで回避できます。

**方法A（かんたん）: 右クリックで開く**
1. `ShelfDrop.app` を**右クリック**
2. 「**開く**」を選択
3. 「開発元を確認できないため…」ダイアログで「**開く**」をクリック

**方法B: ターミナルを使う**
```bash
xattr -rd com.apple.quarantine /Applications/ShelfDrop.app
```
上記コマンドをターミナルに貼り付けて実行後、通常通りダブルクリックで起動できます。

## 初回セットアップ

### 自動ペースト機能を使う場合（任意）

「⚡️ 自動ペースト」ボタンを使うには、**アクセシビリティ権限**が必要です。

1. 自動ペーストボタンをクリックすると、macOS から権限確認ダイアログが表示されます
2. **「システム設定を開く」** をクリック
3. **プライバシーとセキュリティ → アクセシビリティ** を開く
4. `ShelfDrop` のスイッチを **ON** にする
5. これ以降、自動ペーストが使えるようになります

> 💡 自動ペーストを使わない場合、「📋 コピー」ボタンでクリップボードにコピーして、手動で Cmd+V でペーストすることもできます。

## 使い方

| 操作 | 機能 |
|------|------|
| マウスを素早く左右に振る | 棚を呼び出す |
| ファイルを棚にドラッグ | ファイルを保存 |
| 棚のファイルをドラッグ | ファイルを別アプリへドロップ |
| `Control` キーを2回連打 | プロンプト棚を呼び出す |
| プロンプトにカーソルを乗せる | 全文プレビュー（スクロール可能）|
| `Cmd+V` を棚の上で押す | クリップボードの内容を棚に追加 |
