Features

作業の邪魔をしない
新しいクリップボード

シンプルな操作で、ファイルやテキストを即座に管理

🖱️

マウスシェイクで呼び出し

画面の端でカーソルを素早く左右に振るだけ。どのアプリを使っていても瞬時に棚が出現します。

📁

ファイル棚

画像・PDF・動画・どんなファイルもドラッグ&ドロップで一時保存。必要な時にすぐ取り出せます。

📝

プロンプト棚

AIへのプロンプトや定型文をコレクション。ワンクリックでコピー＆自動ペーストが完了します。

⚡️

自動ペースト

棚をしまう → フォーカスが前のアプリへ → 自動でCMD+Vを送信。キーボードを触らず完結します。

✨

複数の棚を管理

用途別に棚を複数作成。プロジェクトごとに分けて整理できます。アプリ終了後も内容を保持します。

🔍

全文プレビュー

長いプロンプトにカーソルを乗せると、横にスクロール可能な全文プレビューがポップアップします。

Install

インストール方法

3ステップで完了します

1

ダウンロードして解凍

ShelfDrop.zip をダウンロードし、ダブルクリックで解凍します。

2

アプリケーションフォルダへ移動

ShelfDrop.app を /アプリケーション フォルダへドラッグします。

3

「開けない」と言われたら右クリックで開く

macOS Gatekeeperによる確認が出ることがあります。その場合は 右クリック → 開く → それでも開く で起動できます。

またはターミナルで以下を実行（コピーしてペーストしてください）：

xattr -rd com.apple.quarantine /Applications/ShelfDrop.app
